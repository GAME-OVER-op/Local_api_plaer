# Планшет Плеер + media-server

Проект из двух частей:

1. **Приложение (`app/`)** — Android-плеер для старого планшета (Android 4.4.2, API 19). Воспроизводит видео по сети через **libVLC**, умеет перематывать и скачивать файлы.
2. **Сервер (`server/`)** — Rust-бинарник `media-server` для другого (рутованного) устройства. Отдаёт содержимое `/storage/emulated/0` по HTTP.

## Приложение

- При запуске спрашивает IP и порт сервера (по умолчанию порт `10930`), хранит историю подключений.
- Файловый проводник с глобальным поиском и обновлением свайпом вниз (pull-to-refresh).
- Плеер на libVLC:
  - векторные иконки кнопок (play/pause, ±10 сек, +1:30, соотношение сторон, на весь экран);
  - перемотка горизонтальным свайпом (пропорционально, с подсказкой);
  - двойной тап: центр — пауза/воспроизведение, левая треть — −10 сек, правая — +10 сек;
  - одиночный тап — показ/скрытие панели управления;
  - скорость воспроизведения, соотношение сторон, иммерсивный полноэкранный режим;
  - экран не гаснет во время воспроизведения.
- Настройки (шестерёнка на главном экране): переключатель светлой/тёмной темы (по умолчанию светлая).
- Все экраны свободно поворачиваются; плавные анимации переходов.

## Сервер

Запуск вручную:

```sh
./media-server -H 0.0.0.0 -p 10930 -r /storage/emulated/0
```

API:

- `GET /` и `GET /health` — проверка (`media-server ok`).
- `GET /list?path=REL` — содержимое папки.
- `GET /search?q=TEXT` — рекурсивный поиск (до 500 результатов).
- `GET /download?path=REL` — скачивание с поддержкой Range (перемотка).

Без авторизации — рассчитано на доверенную локальную сеть.

## Сборка
